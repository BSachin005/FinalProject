Addendum

In creating "Revenge of Lights Out," I aimed to enhance the Lights Out game with additional features, styling, and user interaction. Below are some insights into the creative choices, challenges faced, and solutions implemented during the development process.

Design Choices:

- Colors: I selected a light green background color to create a visually pleasing and calming environment for the game.
- Layout: Using flexbox, I centered the game container both vertically and horizontally to improve the overall presentation.

Game Features:

- Timer: To add an element of challenge, I included a timer that starts when the player clicks the "Start Game" button and stops when the game is successfully completed.
- New Game Button: The "New Game" button resets the board and timer, allowing users to restart the game at any time.

Challenges and Solutions:

- Styling: Ensuring a visually appealing design while maintaining responsiveness was a challenge. The use of flexbox and carefully chosen colors helped create an aesthetically pleasing layout.
- Timer Implementation: Integrating a timer required careful handling of intervals. The startTimer and resetTimer functions manage the timer's state, ensuring accuracy.
- User Interaction: Balancing user interactivity with game functionality was key. The "Start Game" and "New Game" buttons provide a clear and intuitive way for users to control the game.

Insights:

- Structuring the code modularly, using functions for specific tasks, improved code readability and maintenance.
- Continuous testing and iteration were essential for refining the game's features and ensuring a smooth user experience.
- Incorporating user feedback during the development process helped in identifying areas for improvement.
